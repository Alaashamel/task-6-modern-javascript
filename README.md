# Task 6 - Modern JavaScript Concepts (ES6+)

A collection of exercises demonstrating modern JavaScript (ES6+) features.

## Topics Covered

| Task | Topic | Description |
|------|-------|-------------|
| 1 | Strict Mode | Enabling strict mode for better error handling |
| 2 | let & const | Variable declarations with block scope |
| 3 | Default Parameters | Function parameters with default values |
| 4 | Template Literals | String interpolation with backticks |
| 5 | Object Destructuring | Extracting object properties |
| 6 | Array Destructuring | Extracting array elements |
| 7 | Spread Operator (Array) | Merging arrays |
| 8 | Spread Operator (Object) | Merging objects |
| 9 | this Keyword | Object method context |
| 10 | Arrow Function | Concise function syntax |
| 11 | Array filter() | Filtering array elements |
| 12 | Array map() | Transforming array elements |
| 13 | Array reduce() | Reducing arrays to single value |
| Bonus | Combined Exercise | All concepts together |
